How to Use

Clone the repository:

git clone https://github.com/stalwart-coder/CoreJavaPrograms.git

Navigate to the desired folder.

Open the .java files in your preferred IDE or text editor.

Compile and run the programs using:

javac FileName.java
java FileName

Prerequisites

Java Development Kit (JDK): Make sure you have JDK installed on your system. You can download it from Oracle or OpenJDK.

IDE (Optional): IntelliJ IDEA, Eclipse, or VS Code can enhance your development experience.
